Por que o Trabalho Decente é a Chave para o Desenvolvimento Sustentável?

O trabalho decente garante oportunidades iguais, salários justos e condições seguras para todos. Ele promove inclusão social, reduz desigualdades e impulsiona o crescimento econômico sustentável, criando uma base sólida para o desenvolvimento de uma sociedade mais justa e próspera.
